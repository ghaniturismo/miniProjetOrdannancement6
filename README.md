# mini Projet Ordannancement sujet 6

Les heuristiques appartiennent à deux principaux groupes : 

	- les méthodes de liste : Principe : des priorités sont définies sur les tâches. Un ordonnancement est construit de façon itérative en considérant à chaque étape la première machine libre et en lui affectant la tâche disponible de plus grande priorité. (Les priorités peuvent être calculées une fois pour toutes, ou ajustées dynamiquement en cours de l'algo.)
	
	- les techniques de couplage : exemple : couplage de CAMERINI
